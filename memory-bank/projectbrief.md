# Labor Code Portal - Project Brief

## Project Overview

Government portal backend for the Labor Code of the Republic of Uzbekistan with trilingual support (Uzbek, Russian, English).

## Core Requirements

### Functional Requirements
1. **Content Management** - Store and manage Labor Code articles with translations
2. **User Authentication** - Registration, login, 2FA for admins
3. **Comments System** - User comments with moderation
4. **Search** - Full-text search across articles
5. **Chatbot** - AI assistant for finding relevant articles
6. **Analytics** - Statistics on usage

### Non-Functional Requirements
1. **Security** - Government-grade security
2. **Performance** - Response time < 200ms
3. **Scalability** - Handle thousands of concurrent users
4. **Availability** - 99.9% uptime

## Technology Stack
- Laravel 10.x (PHP 8.2+)
- PostgreSQL 15
- Redis
- Laravel Sanctum

## Key Features
- Trilingual content (UZ/RU/EN)
- Role-based access control (Admin/Moderator/User)
- Two-factor authentication
- Comment moderation
- Full-text search
- Activity logging

## Project Scope
- REST API only (no frontend)
- Production-ready with security hardening
- Comprehensive test coverage
